# Lunzy (LNZ) – Community-Driven Meme Token

**Tagline:** Entertainment, Community, and the Journey – A Meme Token Growing Together with Its Community

---

## Project Overview
Lunzy is a community-driven meme cryptocurrency built on the **Avalanche C-Chain (AVAX)**.  
Designed with a fixed supply and an **immutable smart contract**, Lunzy focuses on **transparency, security, and long-term sustainability**.  

Instead of relying on unrealistic promises, Lunzy grows together with its community, fostering creativity, participation, and collective development.

---

## Key Features

- **Immutable & Ownerless:** No privileged roles, no minting, no admin control.  
- **Zero Transaction Fees:** Fully tax-free wallet-to-wallet and DEX transactions.  
- **Security-Oriented:** ERC20 standard.
